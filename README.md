# Zero Panel

Zero Panel is a standalone Elder Scrolls Online utility panel with reorderable buttons, assistant shortcuts, group tools, and fully custom buttons.

## Installation

1. Install the required dependencies:
   - `LibAddonMenu-2.0`
   - `LibAddonMenuOrderListBox`
2. Copy this `Zero_Panel` folder into `Documents\Elder Scrolls Online\live\AddOns`.
3. Enable the add-on in the ESO add-on menu.

## Slash Commands

| Command | What it does |
| --- | --- |
| `/zp` | Open Zero Panel settings. |
| `/zp unlock` | Unlock the panel so it can be dragged to a new position. |
| `/zp lock` | Lock the panel in place after moving it. |
| `/zp reset` | Reset the panel position to its default anchor. |
